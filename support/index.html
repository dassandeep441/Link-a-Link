Generating a support report

JSON diagnostic report for our support team — privacy-safe.

Support page — generate diagnostic reports

How to generate

  1. Open Link-a-Link admin → Support
  2. Click "Generate Support Report"
  3. Download the JSON file
  4. Attach it to your support ticket

What's in the report

The support report contains only aggregate counts and technical metadata:

  • Plugin version and instance ID
  • Total mapping records
  • Aggregate search counts
  • Configuration snapshot (system names only)
  • Recent error counts
ℹ Privacy-safe

Support reports never contain page content, search queries, user account IDs, page titles, or any personal data.

Contacting support

Reach our team through the JSM customer portal.

Steps

  1. Generate a support report (see previous page)
  2. Click "Contact Support" in the admin panel
  3. You'll be redirected to our JSM customer portal
  4. Select the request type that matches your issue
  5. Fill in the details and attach your support report JSON
  6. Submit

Request types

  • Bug Report — something isn't working
  • Feature Request — suggest a new capability
  • Configuration Help — setup assistance
  • Data Issue — mapping data problems
  • Account / Billing — license queries
  • General Question — anything else
