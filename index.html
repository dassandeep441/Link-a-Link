Primary Feature

IntellinQs — Centralized Link Control

Define a base URL once. Reference it everywhere with a short prefix. When the base changes, every reference updates automatically.

Define Base URL Once

Configure a system name, prefix, and base URL in the admin panel. All references resolve through this single configuration point.

Use Prefix Anywhere

Authors type a prefix and document ID (e.g., PDL-123) using the Confluence macro. The full URL generates dynamically on every page render.

Instant Propagation

Change the base URL once in admin. Every page across your instance reflects the new URL immediately — zero manual edits.
